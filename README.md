# 🏦 Banking Business Performance Dashboard

## 📌 Project Overview

The **Banking Business Performance Dashboard** is an end-to-end data analytics project built to analyze customer, loan, and transaction data from a banking dataset. The project demonstrates how raw business data can be transformed into actionable insights using SQL, Microsoft Excel, and Power Query.

The primary objective was to automate data preparation, monitor key business metrics, and build an interactive dashboard that enables business users to make informed decisions quickly.

---

## 🎯 Business Problem

Banks generate large volumes of customer and transaction data every day. Without a centralized reporting system, identifying business trends and monitoring performance becomes time-consuming and error-prone.

This project addresses these challenges by:

- Automating data cleaning and transformation
- Creating interactive KPI dashboards
- Tracking customer and loan performance
- Simplifying recurring business reporting
- Supporting data-driven decision-making

## 🔄 Project Workflow

### 1. Data Extraction

- Imported banking data into Excel
- Loaded tables using Power Query

### 2. Data Cleaning

Performed extensive cleaning including:

- Removing duplicate records
- Handling missing values
- Standardizing text fields
- Correcting data types
- Creating calculated columns
- Formatting dates
- Data validation

---

### 3. Data Transformation

Using **Power Query**, automated:

- Data merging
- Data filtering
- Column transformations
- Derived business metrics
- ETL workflow

---

### 4. SQL Analysis

Used SQL to analyze:

- Customer distribution
- Loan performance
- Transaction trends
- Branch-wise performance
- Customer segmentation

---

### 5. Dashboard Development

Built an interactive Excel dashboard featuring:

- KPI Cards
- Interactive Charts
- Dynamic Filters
- Trend Analysis
- Business Performance Monitoring
